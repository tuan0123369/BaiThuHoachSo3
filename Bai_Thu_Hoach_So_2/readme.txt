Mô tả bài tập: Thiết kế 1 website sử dụng datatable.


Hướng dẫn cài đặt:

1. Vào link: https://datatables.net/examples/basic_init/zero_configuration.html

2. Tạo mới 1 file html trong thư mục xampp htdocs (hoặc www nếu dùng wampp)

3. Ở link ở bước 1 copy link css: https://cdn.datatables.net/1.11.5/css/jquery.dataTables.min.css
và đưa vào header file html: <link rel="stylesheet" href="https://cdn.datatables.net/1.11.5/css/jquery.dataTables.min.css">

4. Tương tự bước 3 với 2 link javascript: 
- https://code.jquery.com/jquery-3.5.1.js
-> <script src="https://code.jquery.com/jquery-3.5.1.js"></script>

- https://cdn.datatables.net/1.11.5/js/jquery.dataTables.min.js
- <script src="https://cdn.datatables.net/1.11.5/js/jquery.dataTables.min.js"> </script>

5. Copy table đã được tạo sẵn ở link bước 1 vào body của file html.
